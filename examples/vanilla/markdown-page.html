# Documentation example

This is a minimal example of the markdown viewer pattern. Wrap any parsed-markdown HTML in <article class="markdown"> and the typographic atoms cascade in.

Tip

Pair this template with marked.js or markdown-it for live rendering of user-authored content.

# Installation

Add the two CSS files and you're done:

<link rel="stylesheet" href="jds/tokens.css">
<link rel="stylesheet" href="jds/index.css">

# Usage

  1. Set data-theme on <html>.
  2. Use the component classes from the catalog.
  3. For markdown content, wrap in .markdown.
